# tiny-ott

My first attempt at a basic Rust implementation of Observational Type Theory by trying to implement Pujet and Tabareau, [Observational Equality: Now for Good](https://dl.acm.org/doi/pdf/10.1145/3498693).

OTT is cool because equality computes by recursion on the type instead of being inductively generated by `refl`, so function extensionality and proof irrelevance fall out as theorems rather than axioms while transport stays definitional.

```bash
cargo build
```

```bash
cargo run -- repl
```

```text
tiny-ott REPL. Type :? for help, :q to quit.
> def id : (A : Type) -> A -> A := \A x => x
def id
  : (A : Type) -> A -> A
  := \A => \x => x
> :t id Nat
id Nat : Nat -> Nat
> id Nat (suc zero)
suc zero
  : Nat
> :l examples/prelude.ott
> :t add 2 3
add 2 3 : Nat
> add 2 3
suc (suc (suc (suc (suc zero))))
  : Nat
> add-zero-id
\_ => _
  : Nat -> Eq Nat _ _
> subst Nat (\n => Nat) zero zero refl zero
zero
  : Nat
> coh Nat Nat refl zero
_
  : Eq Nat zero zero
> :q
```

| command     | effect                                 |
| ----------- | -------------------------------------- |
| `:t <expr>` | infer the type of `<expr>`             |
| `:l <file>` | load definitions from a file           |
| `:?`        | show help                              |
| `:q`        | quit                                   |
| `<decl>`    | run a `def` / `eval` / `check` decl    |
| `<expr>`    | evaluate to normal form and print type |
